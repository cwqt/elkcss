How to use elk-css

  • Download elk.css from this repo
  • Move it to your project folder
  • Include it in the head via (assuming your stylesheet + html file are in the same directory)
<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" type="text/css" href="elk.css">

some stuff

</head>
<body>

your code here
